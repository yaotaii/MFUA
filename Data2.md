FMC (Fixed Mobile Convergence, FMC) КОНВЕРГЕНЦИЯ СТАЦИОНАРНОЙ И МОБИЛЬНОЙ СВЯЗИ

1. Оказание услуг через изолированные друг от друга сети фиксиро-ванной и мобильной связи

Телефоны -> Антенна -> Mobile Servivices -> INTERNET
Домашние устройства -> Роутер -> Свитч -> Fixnet Services -> INTERNET 

2. Совместное использование фиксированной и мобильной сетью об-щего транспорта

Телефоны -> Антенна ->                        | Common    |   ->  Mobile Servivices -> INTERNET
Домашние устройства -> Роутер -> Свитч ->     | transport |   ->   Fixnet Services -> INTERNET 
                                                    ^
                                                    |
                                                    Мост между двумя сетями
